Build a function mergeSort that takes in an array and returns a sorted array, 
using a recursive merge sort methodology.

Pseudo code:

mergeSort (array)

    if array.length <= 1, return the array
    sorted first half = mergeSort(array first half)
    sorted second half = mergeSort(array second half)

    sorted array = merge(first half and second half)
    return sorted array 

merge (array1, array2)

    if both arrays empty, return empty array 
    if array1 empty, return array 2
    if array2 empty, return array 1

    if array1[0] < array2[0]
        return array1[0] glued with merge (rest of array1, array2)
    else 
        return array2[0] glued with merge(array1, rest of array2)
